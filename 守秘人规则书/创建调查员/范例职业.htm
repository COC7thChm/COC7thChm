范例职业

 下面列出的职业只是一些可能的示例（混沌元素的《调查员手册》会提供更多细节）。创造此处未列出的职业时，可以将它们作为参考。洛夫克拉夫特的故事中的重要职业将被标记为 [ 洛式 ]，而标为 [ 现代 ]的职业只适用于现代背景的游戏。

若要创造一个新职业，本职技能的数量不应超过八个，否则职业作为知识和能力（技能）集中体现的概念将变得毫无意义。有些职业只能存在于特定背景，比如黑客只存在于现代。你应该只选择那些符合时代背景的技能。如果你无法确定这一点，就和你的守秘人谈谈。

技能的定义见第四章：技能。

文物学家 [ 洛式 ]：估价，一项社交技能 ( 话术，恐吓，取悦，说服 )，历史，图书馆使用，艺术 / 手艺 ( 任一 )，其他语言，侦查，任意一项其他技能。
信用评级：30–70
本职技能点：EDU×4

艺术家：一项社交技能 ( 话术，恐吓，取悦，说服 )，历史或博物学，心理学，艺术/手艺(任一)，其他语言，侦查，任意两项其他技能。
信用评级：9–50
本职技能点：EDU×2 ＋ POW 或 DEX×2

运动员：格斗 ( 斗殴 )，一项社交技能 ( 话术，恐吓，取悦，说服 )，攀爬，骑术，跳跃，投掷，游泳，任意一项其他技能。
信用评级：9–70
本职技能点：EDU×2 ＋ DEX 或 STR×2

作家 [ 洛式 ]：博物学或神秘学，历史，图书馆使用，心理学，艺术 ( 文学 )，其他语言，母语，任意一项其他技能。
信用评级：9–30
本职技能点：EDU×4

神职人员：一项社交技能 ( 话术，恐吓，取悦，说服 )，会计，历史，聆听，图书馆使用，心理学，其他语言，任意一项其他技能。
信用评级：9–60
本职技能点：EDU×4

罪犯：一项社交技能 ( 话术，恐吓，取悦，说服 )，潜行，心理学，侦查，并从以下技能中选择四项作为专长：格斗，估价，机械维修，妙手，乔装，射击和锁匠。
信用评级：5–65
本职技能点：EDU×2 ＋ DEX 或 STR×2

业余艺术爱好者 [ 洛式 ]：一项社交技能 ( 话术，恐吓，取悦，说服 )，骑术，射击，艺术 / 手艺 ( 任一 )，其他语言，任意三项其他技能。
信用评级：50–99
本职技能点：EDU×2 ＋ APP×2

医生 [ 洛式 ]：急救，科学 ( 生物学 )，科学 ( 药学 )，心理学，医学，其他语言 ( 拉丁语 )，任意两项其他技能作为学术或个人专长 ( 例如精神病学家可能会选择精神分析 )。
信用评级：30–80
本职技能点：EDU×4

流浪者：导航，一项社交技能 ( 话术，恐吓，取悦，说服 )，聆听，攀爬，潜行，跳跃，任意两项其他技能。
信用评级：0–5
本职技能点：EDU×2 ＋ APP，DEX 或 STR×2

工程师：操作重型机械，电气维修，机械维修，科学 (工程学 )，科学 ( 物理学 )，图书馆使用，艺术 / 手艺 ( 工程制图 )，任意一项其他技能。
信用评级：30–60
本职技能点：EDU×4

艺人：两项社交技能 ( 话术，恐吓，取悦，说服 )，聆听，乔装，心理学，艺术/手艺(表演)，任意两项其他技能。
信用评级：9-70
本职技能点：EDU×2 ＋ APP×2

农民：博物学，操作重型机械，一项社交技能 ( 话术，恐吓，取悦，说服)，机械维修，汽车驾驶(或运货马车)，艺术 / 手艺 ( 耕作 )，追踪，任意一项其他技能。
信用评级：9–30
本职技能点：EDU×2 ＋ DEX 或 STR×2

黑客 [ 现代 ]：电气维修，电子学，一项社交技能 ( 话术，恐吓，取悦，说服 )，计算机使用，图书馆使用，侦查，任意两项其他技能。
信用评级：10–70
本职技能点：EDU×4

记者[洛式]：一项社交技能 (话术，恐吓，取悦，说服)，历史，母语，图书馆使用，心理学，艺术 / 手艺 ( 摄影 )，任意两项其他技能。
信用评级：9–30
本职技能点：EDU×4

律师：心理学，图书馆使用，会计，两项社交技能 ( 话术，恐吓，取悦，说服 )，法律，任意两项其他技能。
信用评级：30–80
本职技能点：EDU×4

图书馆管理员 [ 洛式 ]：会计，母语，图书馆使用，其他语言，任意四项其他技能作为个人专长或专业阅读主题。
信用评级：9–35
本职技能点：EDU×4

军官：导航，两项社交技能 ( 话术，恐吓，取悦，说服 )，会计，射击，生存，心理学，任意一项其他技能。
信用评级：20–70
本职技能点：EDU×2 ＋ DEX 或 STR×2

传教士：博物学，一项社交技能 ( 话术，恐吓，取悦，说服 )，机械维修，急救，医学，艺术 / 手艺，任意两项其他技能。
信用评级：0–30
本职技能点：EDU×4

音乐家：一项社交技能 ( 话术，恐吓，取悦，说服 )，聆听，心理学，艺术/手艺(乐器)，任意四项其他技能。
信用评级：9–30
本职技能点：EDU×2 ＋ DEX 或 POW×2

超心理学家：历史，人类学，神秘学，图书馆使用，心理学，艺术 / 手艺 ( 摄影 )，其他语言，任意一项其他技能。
信用评级：9–30
本职技能点：EDU×4

飞行员：操纵 ( 飞行器 )，操作重型机械，导航，电气维修，机械维修，科学 ( 天文学 )，任意两项其他技能。
信用评级：20–70
本职技能点：EDU×2 ＋ DEX×2

警探 [ 洛式 ]：法律，一项社交技能 ( 话术，恐吓，
取悦，说服 )，聆听，射击，心理学，艺术 / 手艺 ( 表
演 ) 或乔装，侦查，任意一项其他技能。
信用评级：20–50
本职技能点：EDU×2 ＋ DEX 或 STR×2

警察：法律，格斗 ( 斗殴 )，一项社交技能 ( 话术，恐吓，取悦，说服 )，急救，射击，心理学，侦查，及以下任意一项技能作为个人专长：汽车驾驶或骑术。
信用评级：9–30
本职技能点：EDU×2 ＋ DEX 或 STR×2

私家侦探：法律，一项社交技能 ( 话术，恐吓，取悦，说服 )，乔装，图书馆使用，心理学，艺术 / 手艺 ( 摄影 )，侦查，及任意一项其他技能 ( 例如计算机使用，射击，锁匠 )。
信用评级：9–30
本职技能点：EDU×2 ＋ DEX 或 STR×2

教授 [ 洛式 ]：母语，图书馆使用，心理学，其他语言，任意四项技能作为学术或个人专长。
信用评级：20–70
本职技能点：EDU×4

士兵：格斗，攀爬或游泳，潜行，闪避，射击，生存，及以下任意两项技能 : 机械维修，急救或其他语言。
信用评级：9–30
本职技能点：EDU×2 ＋ DEX 或 STR×2

部落成员：博物学，格斗或投掷，聆听，攀爬，神秘学，生存 ( 任一 )，游泳，侦查。
信用评级：0–15
本职技能点：EDU×2 ＋ DEX 或 STR×2

狂热者：两项社交技能 ( 话术，恐吓，取悦，说服 )，历史，潜行，心理学，及任意三项其他技能。
信用评级：0–30
本职技能点：EDU×2 ＋ APP 或 POW×2
